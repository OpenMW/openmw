Stereo View Settings
####################

eye resolution x
----------------

:Type:		integer
:Range:		> 0
:Default:	3128

The default values are based on an HP Reverb G2 HMD.

eye resolution y
----------------

:Type:		integer
:Range:		> 0
:Default:	3060

The default values are based on an HP Reverb G2 HMD.

left eye offset x
-----------------

:Type:		double precision floating point
:Range:		any
:Default:	-2.35

Left eye offset from center, expressed in MW units (1 meter = ~70).

left eye offset y
-----------------

:Type:		double precision floating point
:Range:		any
:Default:	0.0

Left eye offset from center, expressed in MW units (1 meter = ~70).

left eye offset z
-----------------

:Type:		double precision floating point
:Range:		any
:Default:	0.0

Left eye offset from center, expressed in MW units (1 meter = ~70).

left eye orientation x
----------------------

:Type:		double precision floating point
:Range:		-1.0 to 1.0
:Default:	0.0

Left eye orientation, expressed as a quaternion.

left eye orientation y
----------------------

:Type:		double precision floating point
:Range:		-1.0 to 1.0
:Default:	0.0

Left eye orientation, expressed as a quaternion.

left eye orientation z
----------------------

:Type:		double precision floating point
:Range:		-1.0 to 1.0
:Default:	0.0

Left eye orientation, expressed as a quaternion.

left eye orientation w
----------------------

:Type:		double precision floating point
:Range:		-1.0 to 1.0
:Default:	1.0

Left eye orientation, expressed as a quaternion.

left eye fov left
-----------------

:Type:		double precision floating point
:Range:		-π to π
:Default:	-0.86

Left eye field of view, expressed in radians.

left eye fov right
------------------

:Type:		double precision floating point
:Range:		-π to π
:Default:	0.78

Left eye field of view, expressed in radians.
